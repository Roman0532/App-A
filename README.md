## Инструкция по сборке и запуску приложения

     Для сборки проекта используйте команду ./BUILD.sh

     Для запуска проекта используйте команду ./RUN.sh

     Для запуска тестов используйте команду LOGIN=sa PASSWORD= ./TEST.sh
